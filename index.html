Project Description

A distributed library management system implemented in Java using Socket Programming. The system supports multiple backend servers, request routing, replication and leader election.

Implemented Features

  • Client / Server Architecture
  • Load Balancer
  • Sticky Session
  • Round Robin Distribution
  • Health Check
  • Heartbeat Monitoring
  • Fault Tolerance
  • Replication Between Servers
  • Bully Leader Election
  • Serialization
  • Pass By Value
  • Distributed Storage

System Components

  • BookClient
  • LoadBalancer
  • BookServer
  • BookService
  • BookStoreFile
  • Shared Request / Response

Technologies

  • Java
  • Socket Programming
  • Multithreading
  • Serialization
  • Distributed Systems Concepts
